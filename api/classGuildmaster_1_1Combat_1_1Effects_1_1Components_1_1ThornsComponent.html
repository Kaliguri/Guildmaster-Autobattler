Detailed Description

Шипы: отражает долю полученного урона обратно атакующему.

Реактивный — слушает CombatEvent.DamageTaken своего носителя (вики «6» §5.5).

Отражённый урон идёт обычным ICombatContext.DealDamage и сам порождает события — взаимные шипы пинг-понгуют, но дренаж очереди капается в CombatSimulation (детерминированно завершается, без рекурсии).
