Detailed Description

Рантайм стат-объект юнита.

Хранит группы модификаторов по источнику, вычисляет итог по формуле (base + ΣFlat) × (1 + ΣPercentAdd) × Π(1 + PercentMult) и кэширует результат с инвалидацией по dirty-флагу (вики «10» §5.2, «11» §1).
