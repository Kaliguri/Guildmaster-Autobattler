Detailed Description

Опциональный шов: компонент объявляет масштабируемую потенцию.

EffectSystem резолвит её из статов источника один раз при наложении и кладёт снимок в RuntimeEffect.ScaledPotency (per-second rate для DoT/HoT — НЕ запечённый total, вики «11» §5.1).
