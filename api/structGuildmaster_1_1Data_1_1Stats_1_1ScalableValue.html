Detailed Description

Величина, масштабируемая от статов источника: Base + Σ(reader.Get(term.Stat) × term.Ratio).

Основа для урона/исцеления способностей (напр. «120 + 0.6×AbilityPower»). Авторится на SO.

Шов закладывается в Фазе 1; реальное применение в способностях — Фаза 2.
