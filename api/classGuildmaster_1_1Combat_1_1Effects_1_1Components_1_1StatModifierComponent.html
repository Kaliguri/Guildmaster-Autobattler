Detailed Description

Накладывает стат-модификаторы на носителя на время жизни эффекта.

Источник снятия — сам RuntimeEffect, поэтому моды уходят разом на OnExpire. При стаках величина линейно умножается на число стаков (вики «6» §5.5).
