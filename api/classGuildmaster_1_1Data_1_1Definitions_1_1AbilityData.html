Detailed Description

Определение активной способности реликвии: эффекты при касте, кулдаун, стоимость ресурса, способ выбора цели.

Сериализуется на RelicData (вики «6» §1, «12» §2.4). Пассивки — это RelicData.GrantedEffects с постоянной длительностью, отдельной сущности не требуют.
