◆ ModifierOp

enum Guildmaster.Data.Stats.ModifierOp

Операция стат-модификатора.

Формула сборки стата: final = (base + ΣFlat) × (1 + ΣPercentAdd) × Π(1 + PercentMult) (вики «11. Стат-система» §1, «6. Боевая модель» §3).

Enumerator
Flat 

Плоская прибавка к базе (складывается).

PercentAdd 

Аддитивный процент: все слагаемые суммируются в одну скобку (1 + Σ).

PercentMult 

Мультипликативный процент: каждый множитель отдельный (1 + x), перемножаются.

◆ StatType

enum Guildmaster.Data.Stats.StatType

Канонический список боевых статов (ровно 30).

Источник правды — вики «11. Стат-система (полный список статов)». Все значения присвоены явно и стабильны: Unity сериализует enum по int, поэтому порядок/номера менять НЕЛЬЗЯ (сломает .asset).

Фаза 1 реально использует 19 статов (авто-атаки без эффектов/способностей/ресурсов) — помечены [Ф1]. Остальные объявлены сразу (список финализирован), но дефолты/клампы/логика для них подключаются в своих фазах.
