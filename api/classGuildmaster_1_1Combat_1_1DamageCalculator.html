Detailed Description

Статический калькулятор урона.

Инкапсулирует всю логику модификации урона: броня, критические удары, уязвимости.

Все методы статические — класс не хранит состояния. Для расширения системы урона создавайте новые методы здесь, а не разносите формулы по юнитам.
