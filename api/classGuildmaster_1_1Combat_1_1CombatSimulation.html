Detailed Description

Детерминированная тиковая симуляция боя.

Реализует ICombatContext — единственная точка мутации состояния боя из систем и (Фаза 2) компонентов эффектов.

Порядок систем за тик: ApplyCommands → Targeting → Movement → SpatialHashRebuild → AutoAttack → Projectiles → Effects → Death → CheckOutcome → currentTick++.

(вики «10» §5.1).
