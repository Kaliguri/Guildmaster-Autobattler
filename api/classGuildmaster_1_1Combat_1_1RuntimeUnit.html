Detailed Description

Рантайм-представление юнита на один бой.

POCO — без MonoBehaviour и ScriptableObject. Создаётся через RuntimeUnitFactory; владеет Stats, текущим HP/ресурсом/щитом и состоянием движения (вики «10» §5.2).
