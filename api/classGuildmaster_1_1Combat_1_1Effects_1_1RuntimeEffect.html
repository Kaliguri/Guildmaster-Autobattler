Detailed Description

Экземпляр эффекта на конкретном юните (POCO, живёт один бой).

Несёт per-unit рантайм- состояние, которого НЕ может быть в общих [SerializeReference]-компонентах EffectData — те шарятся между всеми носителями эффекта и обязаны быть stateless (вики «12» §2.2, «6» §5).
