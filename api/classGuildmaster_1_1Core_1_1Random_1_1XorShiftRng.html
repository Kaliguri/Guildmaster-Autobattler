Detailed Description

Детерминированный PRNG (xorshift128).

Одинаковый сид → одинаковая последовательность на любой платформе (только целочисленные операции).

Начальное состояние выводится из одного ulong-сида через SplitMix64, чтобы даже «слабые» сиды (0, 1, …) давали хорошо размешанное состояние.
