Detailed Description

Равномерная сетка для пространственных запросов без аллокаций на горячем пути.

Используется для QueryRadius в TargetingSystem и AOE-эффектах (Фаза 2).

Не влияет на геометрию поля — оно по-прежнему непрерывное (вики «10» §5.3).
