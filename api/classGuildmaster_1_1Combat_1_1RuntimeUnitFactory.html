Detailed Description

Единственная точка сборки RuntimeUnit из SO-данных.

Шаги сборки (вики «10» §5.2, «6» §3): дефолты из StatsConfig → моды реликвии → таланты сосуда → пассивки (RelicData.GrantedEffects с постоянной длительностью) → активки (AbilityRuntime) → ресурс (StatType.StartResource) → CurrentHP = Get(MaxHP).

Пассивки применяются до инициализации CurrentHP: пассив на +MaxHP должен поднять и стартовое здоровье. Поэтому фабрике нужны EffectSystem и боевой контекст — наложение пассивки зовёт OnApply компонентов (вики «12» §6, шаг 9).
