Detailed Description

Мост «симуляция → презентация».

Подписывается на C#-события CombatSimulation, спавнит/деспавнит UnitView и ретранслирует события в MessagePipe, чтобы Audio/VFX/UI подписывались независимо от симуляции (вики «10» §7, стек — CLAUDE.md). Только читает состояние симуляции — никогда не мутирует.
