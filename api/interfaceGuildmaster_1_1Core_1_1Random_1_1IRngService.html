Detailed Description

Детерминированный источник случайности симуляции.

Любой рандом в бою идёт ТОЛЬКО через этот интерфейс — это одно из условий детерминизма (см. вики «10. Архитектура реализации — Фаза 1» §5.1, «5. Стек и архитектура»).

Реализация — XorShiftRng. На каждый бой создаётся свой инстанс с суб-сидом (runSeed + battleIndex + attempt) — фабрика в боевом scope (Фаза 1, шаг 9).
