Detailed Description

Рантайм-контракт поведения эффекта.

Производный от Data-маркера IEffectComponent: сериализационный якорь живёт в Data, а хуки, оперирующие боевым состоянием (EffectContext), — здесь, в Combat (кросс-сборочный шов, вики «10» §2.2, «12» §3.1).

Компоненты stateless: экземпляр шарится между всеми носителями эффекта. Per-unit состояние — в RuntimeEffect/EffectContext. OnApply обязан быть идемпотентен при re-apply на стак/рефреш (EffectSystem зовёт OnExpire→OnApply).
