Detailed Description

Базовый контракт команды симуляции — упорядочивающий ключ для детерминированного применения на границе тика.

Все мутации сим во время боя входят через команды: это шов host-authoritative мультиплеера и реплеев (см. вики «10. Архитектура» §5.6).

Конкретное исполнение (Apply(CombatSimulation)) добавляет слой Combat, расширяя этот контракт — Core не знает о симуляции и не зависит от неё вверх.
