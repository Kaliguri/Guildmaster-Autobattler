Detailed Description

Контекст, передаваемый компоненту эффекта (in).

Даёт доступ к цели, источнику, шву влияния на мир (ICombatContext), снимку потенции для текущего компонента и числу стаков. Так компоненты остаются stateless и юнит-тестируемы с мок-контекстом (вики «6» §5.1, «12» §3.2).
