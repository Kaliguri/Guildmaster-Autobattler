Detailed Description

Глобальный шаблон стат-системы: дефолты статов + тюнинг-константы пайплайна (armor-константа, клампы скорости атаки).

Единственный экземпляр на проект — база сборки любого юнита (вики «10» §4.2, «11» §3–§4).
