Detailed Description

World-space визуальное представление RuntimeUnit.

Интерполирует позицию между тиками (сим 30 Hz, рендер 60+ fps). Feel-хуки — UnityEvent поля: Phase 1 пусты (no-op), в Inspector подключается MMF_Player.PlayFeedbacks без compile-time зависимости. (вики «10» §7).
