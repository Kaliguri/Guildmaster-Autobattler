Detailed Description

Тик способностей: убывание кулдаунов и детерминированный плейсхолдер-каст готовых активок (полный выбор «когда/что кастовать» — AI Фазы 3, вики «12» §1, §9).

Каст списывает ресурс, ставит кулдаун (× CooldownEff) и накладывает эффекты способности на цель.
