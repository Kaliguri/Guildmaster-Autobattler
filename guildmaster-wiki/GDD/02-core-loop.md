# 02 — Игровой цикл (Core Loop)

**Статус:** 🟡 Draft

---

## Основной цикл

```
[Гильдия] → [Подготовка] → [Бой] → [Результат] → [Гильдия]
```

### 1. Гильдия (между боями)
- Нанять/уволить героев
- Купить экипировку и расходники
- Улучшить здания гильдии
- Выбрать следующее задание

### 2. Подготовка (перед боем)
- Расставить юнитов на поле
- Назначить экипировку
- Задать приоритеты целей (опционально)

### 3. Бой (автобатлер)
- Юниты действуют по своим правилам
- Игрок наблюдает, не управляет напрямую
- Продолжается до победы одной из сторон

### 4. Результат
- Получить награду (золото, предметы, опыт)
- Потерять/получить героев (permadeath?)
- Разблокировать новый контент
