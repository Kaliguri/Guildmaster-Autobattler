Краткий вход в контентные заметки по реликвиям героев.

## Что такое реликвии

Реликвии — основа боевой роли «Сосуда»: задают профиль статов, пассивки и активки.
Здесь фиксируются черновые ГД-идеи по реликвиям-классам (по одной карточке на класс).

## Связь с движком

Часть механик карточек требует движковых расширений, которых нет в библиотеке Фазы 2 (заряды, атака в движении, хил-автоатака, линия/конус, смещения). Их перечень и ориентир по фазам — [Тех: 6. Боевая модель §10](../../Игровая%20документация%20-%20Техническая%20часть/6.%20Боевая%20модель%20(юниты,%20статы,%20эффекты).md). Реликвии тестируются вместе с AI-движком в Фазе 3 (test-arena lite).
