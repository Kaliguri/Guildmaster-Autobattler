## Классы

> *Какие архетипы юнитов есть в игре?*

| Класс | Роль | Описание |
|---|---|---|
| Воин | Танк | Принимает урон, держит линию |
| Лучник | ДД (дальний бой) | Атакует с безопасной дистанции |
| Маг | ДД (AOE/магия) | Высокий урон, низкое здоровье |
| Жрец | Поддержка | Лечение, баффы союзников |
| Вор | Убийца | Высокий разовый урон, цель — задняя линия |

---

## Базовые характеристики

| Параметр | Описание |
|---|---|
| `HP` | Очки здоровья |
| `Attack` | Урон базовой атаки |
| `AttackSpeed` | Интервал между атаками (сек) |
| `MoveSpeed` | Скорость передвижения по полю |
| `Range` | Дальность атаки (клетки) |
| `Armor` | Снижение физического урона |
| `MagicResist` | Снижение магического урона |

## Поведение AI

Каждый юнит следует простым правилам:
1. Найти цель по приоритету
2. Подойти в зону атаки
3. Атаковать / применить способность

Приоритеты цели:
- [ ] Ближайший враг
- [ ] Враг с наименьшим HP
- [ ] Враг с наибольшим уроном
