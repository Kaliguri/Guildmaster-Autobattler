## Типы урона

| Тип | Против чего | Пример |
|---|---|---|
| Физический | Снижается `Armor` | Атака мечом |
| Магический | Снижается `MagicResist` | Огненный шар |
| Чистый | Не снижается ничем | Яд, кровотечение |

## Статус-эффекты

| Эффект | Описание |
|---|---|
| Stunned | Пропускает ход/действие |
| Poisoned | Урон в конце каждого хода |
| Burning | Физический DoT |
| Slowed | Уменьшение скорости |
| Buffed | Временное усиление характеристики |
